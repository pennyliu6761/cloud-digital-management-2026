### 💡 觀念講解：Mail Merge 是什麼？

**Mail Merge（郵件合併）** 是一種把「模板」和「資料清單」結合，
自動產出 N 份個人化文件的技術。
```
【模板】                     【試算表資料】              【輸出 × N 份】
親愛的 <<姓名>> 同學    +    王小明, 台積電, 3/1     =   親愛的 王小明 同學
實習單位：<<企業名稱>>       李美麗, 聯電, 3/15          實習單位：台積電
開始日期：<<開始日期>>       陳志偉, 鴻海, 4/1           開始日期：3/1
```

> **AutoCrat + Google 試算表 = 雲端版 Mail Merge**
> 隨時可以跑、任何人都能存取結果，不需要安裝任何軟體。

### 💡 觀念講解：公式 = 試算表的自動化程式

很多人把試算表當成「有格線的 Word」，只是打字用的。
**但試算表的真正威力在於公式**：你定義一個規則，它就永遠自動計算，
資料更新時結果也即時更新。
```
一個好的試算表架構 =
「原始資料區」（不修改） + 「公式計算區」（自動產生）
```

#### 公式 1：`DATEDIF` — 計算實習天數

在新欄位 **I2** 輸入：
```
=DATEDIF(E2, F2, "D")
```
**輸出範例：** `61`（天數）

**用途：** 計算合約天數、保固剩餘天數、員工年資。

---

#### 公式 2：`IF` — 判斷是否達到實習最低天數

在新欄位 **J2** 輸入：
```
=IF(I2>=30, "✅ 符合最低天數", "❌ 不足 30 天")
```

=IFS(G2>=40000, "A級", G2>=30000, "B級", G2>=25000, "C級", G2=0, "無薪實習", TRUE, "資料待確認")

在試算表空白區建立統計表(實習類型&人數)

#### 公式 8：`VLOOKUP` — 跨表查找企業聯絡資料

建立第二張工作表「**企業聯絡資料**」：

| A：企業名稱 | B：產業別 | C：聯絡窗口 | D：聯絡 Email |
|-----------|---------|-----------|-------------|
| 台積電 | 半導體 | 王經理 | hr@tsmc.com |
| 聯電 | 半導體 | 李協理 | hr@umc.com |
| 鴻海精密 | 電子製造 | 陳主任 | hr@foxconn.com |
| 聯發科 | IC 設計 | 張副總 | hr@mediatek.com |

回到第一張工作表，在 **L2** 輸入：
```
=IFERROR(VLOOKUP(D2, 企業聯絡資料!A:D, 3, FALSE), "查無聯絡窗口")
```

**公式解析：**
- `D2`：要查找的企業名稱
- `企業聯絡資料!A:D`：去第二張工作表的 A 到 D 欄查
- `3`：回傳第 3 欄（聯絡窗口姓名）
- `FALSE`：精確比對
- `IFERROR(..., "查無聯絡窗口")`：找不到時顯示提示，不顯示難看的 `#N/A`
